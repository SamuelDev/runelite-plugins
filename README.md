# Runelite plugins
This repo holds my runelite plugins. Swap branches to the plugin you want to browse.

## Plugins

### Average Total Level
Displays your average total level on the skill tab. Works with Virtual total levels plugin.

### Hide Prayer Descriptions
Hides the tooltips (or parts of them) for prayers when hovering over them in the prayer tab.

### Better NPC Highlight
A more customizable way of highlighting NPCs. Originally created by [MoreBuchus](https://github.com/MoreBuchus/), transferred to the Runelite maintainers, then to me.
